# 欢迎来到我的博客

这是用 **fuwari** 搭建、并通过 **Pages CMS** 管理的第一篇示例文章。

之后你可以在浏览器里直接写文章：填标题、选日期、加标签、写正文，点保存就会自动提交到 GitHub 并重新部署。

## 小提示

- 设为 `draft: true` 的文章不会进入生产构建。
- 正文支持 Markdown，也支持代码块、提示框（note/tip/warning）等扩展语法。

> 祝你写作愉快！
